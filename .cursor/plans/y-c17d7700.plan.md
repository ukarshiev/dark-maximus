# Исправление импорта YooKassa

## Шаги

1. Проверить актуальную структуру пакета `yookassa` и найти корректный модуль для `ApiError`.
2. Обновить импорт в `src/shop_bot/webhook_server/app.py`, привести обработку ошибок к актуальному API.
3. Запустить `npx nx serve bot` либо повторить контейнерный запуск, убедиться, что бот стартует без `ModuleNotFoundError`.
4. При необходимости обновить документацию в `docs/` и добавить свежую запись в `CHANGELOG.md`, синхронизировать версию в `pyproject.toml`.
